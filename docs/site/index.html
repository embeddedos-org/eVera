eVera

AI-powered virtual assistant with 24+ agents, 183+ tools, 3D animated avatar (Three.js), voice I/O, and rich plugin architecture. Desktop, mobile, and web.

View on GitHub Quick Start

Features

Your intelligent companion across every platform

24+ Autonomous Agents

Specialized agents that handle tasks autonomously — from research to code generation to scheduling.

183+ Tools

Extensive tool library covering file operations, web search, APIs, data processing, and more.

3D Animated Avatar

Beautiful Three.js-powered 3D avatar with lip sync, expressions, and gesture animations.

Voice Input/Output

Natural voice interaction with speech recognition and text-to-speech synthesis.

Rich Plugin Architecture

Extend functionality with a powerful plugin system — build and share custom capabilities.

Desktop (Electron)

Native desktop experience on Windows, macOS, and Linux via Electron framework.

Mobile (React Native)

Full-featured mobile app for iOS and Android built with React Native.

Multi-Model LLM Support

Connect to OpenAI, Anthropic, local models, and more — switch between providers seamlessly.

Web Interface

Access eVera from any browser with a responsive, feature-rich web interface.

Context-Aware Conversations

Maintains conversation history and context for intelligent, coherent multi-turn interactions.

Quick Start

Get up and running in seconds

pip install -e .
evera start
